Project Work

Beyond Helios | Mobile Game Development

Beyond Helios is a mobile game which I independently designed, developed, and published to the Apple App Store. The game is a homage to the arcade machines of old where players compete to see how long they can survive and who can post the highest score. Featuring infinite procedural level generation, cloud game saving, realtime pathfinding, and so much more, Beyond Helios is truly one of my proudest achievements.
